Eveve Booking Widget - Basic Example

This page demonstrates the simplest way to embed the Eveve booking widget into your website.

Zero Installation Required! You don't need to download or host any files on your server. Just add one script tag to your website and the widget will load everything it needs directly from our servers.
Note: Replace 123 with your actual restaurant ID provided by Eveve.

Example Code

Copy and paste this code into your website where you want the booking widget to appear:

<!-- 1. Add a container div where you want the widget to appear -->
<div id="eveve-booking" data-restaurant="123"></div>

<!-- 2. Add this single script tag (preferably just before </body>) -->
<script src="https://booking.eveve.com/form1-0-0/embed.js"></script>
That's it! The script automatically loads all required CSS, JavaScript, and theme files from our servers. You don't need to upload or host any files yourself.

How It Works

When you add the script to your page:

  1. The script automatically finds all widget containers on your page
  2. It loads all required CSS, JavaScript, and theme files from our servers
  3. The widget appears in the container div you specified
  4. All booking data is securely processed through our servers

Additional Options

You can customize the widget by adding data attributes to the container:

<div id="eveve-booking" 
    data-restaurant="123"
    data-theme="brand-roboto"
    data-default-guests="4"
    data-lang="en">
</div>

Available data attributes:

  • data-restaurant: Your Eveve restaurant ID (required)
  • data-theme: Widget theme (default: "light")
  • data-theme-css: Custom theme CSS URL
  • data-default-guests: Pre-selected guest count
  • data-default-date: Pre-selected date (YYYY-MM-DD format)
  • data-lang: Language code (default: "en")
  • data-debug: Enable debug mode ("true" or "false")
