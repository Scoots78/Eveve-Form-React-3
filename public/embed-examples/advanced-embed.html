Eveve Booking Widget - Advanced Examples

This page demonstrates advanced usage of the Eveve booking widget, including multiple widgets on the same page, different themes, and integration with various CMS platforms.

Note: Replace 123, 456, etc. with your actual restaurant IDs provided by Eveve.

Multiple Widgets on the Same Page

You can embed multiple booking widgets on the same page, each with different configurations and themes:

Standard Light Theme

<div class="eveve-widget" 
    data-restaurant="123" 
    data-theme="light"
    data-default-guests="2">
</div>

Brand Roboto Theme

<div class="eveve-widget" 
    data-restaurant="456" 
    data-theme="brand-roboto"
    data-default-guests="4">
</div>
Tip: Each widget operates independently and can have different restaurant IDs, themes, and configurations.

Advanced Configuration Options

Attribute Description Example
data-restaurant or data-est Your Eveve restaurant ID (required) data-restaurant="123"
data-theme Widget theme name data-theme="brand-roboto"
data-theme-css URL to a custom theme CSS file data-theme-css="/custom-themes/my-theme.css"
data-default-guests Pre-selected guest count data-default-guests="4"
data-default-date Pre-selected date (YYYY-MM-DD format) data-default-date="2025-12-31"
data-lang Language code data-lang="en"
data-debug Enable debug mode data-debug="true"

WordPress Integration

Add the widget to your WordPress site using one of these methods:

Method 1: Using a Custom HTML Block

  1. Edit the page where you want to add the booking widget
  2. Add a "Custom HTML" block
  3. Paste the following code:
<div id="eveve-booking" data-restaurant="123" data-theme="brand-roboto"></div>
<script src="https://yourdomain.com/form1-0-0/embed.js"></script>

Method 2: Using a WordPress Plugin

  1. Install and activate the "Insert Headers and Footers" plugin
  2. Go to Settings > Insert Headers and Footers
  3. Add the script to the footer section:
<script src="https://yourdomain.com/form1-0-0/embed.js"></script>
  • Then add the container div to any page using a Custom HTML block:
    • <div id="eveve-booking" data-restaurant="123" data-theme="brand-roboto"></div>

    Method 3: Using a Shortcode

    Add this code to your theme's functions.php file:

    function eveve_booking_shortcode($atts) {
    $atts = shortcode_atts(array(
        'restaurant' => '',
        'theme' => 'light',
        'guests' => '',
        'date' => '',
        'lang' => 'en'
    ), $atts);
    
    $html = '<div class="eveve-widget"';
    foreach($atts as $key => $value) {
        if(!empty($value)) {
            $html .= ' data-' . esc_attr($key) . '="' . esc_attr($value) . '"';
        }
    }
    $html .= '></div>';
    
    return $html;
}
add_shortcode('eveve_booking', 'eveve_booking_shortcode');

// Add the script to the footer
function eveve_booking_script() {
    echo '<script src="https://yourdomain.com/form1-0-0/embed.js"></script>';
}
add_action('wp_footer', 'eveve_booking_script');

    Then use the shortcode in your pages or posts:

    [eveve_booking restaurant="123" theme="brand-roboto" guests="4"]

    Squarespace Integration

    1. Go to the page where you want to add the booking widget
    2. Add a "Code" block
    3. Paste the following code:
    <div id="eveve-booking" data-restaurant="123" data-theme="brand-roboto"></div>
<script src="https://yourdomain.com/form1-0-0/embed.js"></script>

    For site-wide integration:

    1. Go to Settings > Advanced > Code Injection
    2. Add the script to the Footer section:
    <script src="https://yourdomain.com/form1-0-0/embed.js"></script>
  • Then add just the container div to any page using a Code block:
    • <div id="eveve-booking" data-restaurant="123" data-theme="brand-roboto"></div>

    Wix Integration

    1. Go to the page where you want to add the booking widget
    2. Add an "HTML iframe" element
    3. Click "Enter code"
    4. Paste the following code:
    <div id="eveve-booking" data-restaurant="123" data-theme="brand-roboto"></div>
<script src="https://yourdomain.com/form1-0-0/embed.js"></script>

    For advanced integration with Wix Velo (formerly Corvid):

    1. Go to Settings > Custom Code
    2. Add a new script in the "Body - End" section:
    <script src="https://yourdomain.com/form1-0-0/embed.js"></script>
  • Then use the Wix HTML component to add just the container:
    • <div id="eveve-booking" data-restaurant="123" data-theme="brand-roboto"></div>

    Shopify Integration

    1. Go to Online Store > Themes
    2. Click "Actions" > "Edit code"
    3. Add the script to theme.liquid before the closing </body> tag:
    <script src="https://yourdomain.com/form1-0-0/embed.js"></script>

    Then add the container to any page using a Custom HTML section:

    1. Edit the page where you want to add the booking widget
    2. Add a "Custom HTML" section
    3. Paste the following code:
    <div id="eveve-booking" data-restaurant="123" data-theme="brand-roboto"></div>

    Alternatively, you can create a custom page template with the widget:

    1. Go to Online Store > Themes > Actions > Edit code
    2. Create a new template under Templates (e.g., page.booking.liquid)
    3. Add your page content and the widget code:
    {% extends 'page' %}

{% block content %}
  <div class="page-width">
    <div class="grid">
      <div class="grid__item medium-up--one-whole">
        <h1>{{ page.title }}</h1>
        {{ page.content }}
        
        <div id="eveve-booking" data-restaurant="123" data-theme="brand-roboto"></div>
      </div>
    </div>
  </div>
{% endblock %}

    Custom HTML Integration

    For custom websites or HTML pages, simply add the following code where you want the widget to appear:

    <!-- Eveve Booking Widget Container -->
<div id="eveve-booking" data-restaurant="123" data-theme="brand-roboto"></div>

<!-- Eveve Booking Widget Script -->
<script src="https://yourdomain.com/form1-0-0/embed.js"></script>

    For multiple widgets or site-wide integration, add the script once in the footer:

    <!-- In the <head> or before </body> -->
<script src="https://yourdomain.com/form1-0-0/embed.js"></script>

<!-- Then add containers wherever needed -->
<div id="eveve-booking-1" data-restaurant="123" data-theme="light"></div>
<div id="eveve-booking-2" data-restaurant="456" data-theme="brand-roboto"></div>

    Advanced Features

    Programmatic Initialization

    You can manually initialize widgets or reinitialize them after DOM changes:

    // Initialize a specific container
const container = document.getElementById('my-booking-widget');
window.EveveWidget.init(container);

// Initialize all widgets on the page
window.EveveWidget.initAll();

    Dynamic Widget Creation

    Create widgets dynamically with JavaScript:

    // Create a new widget container
const container = document.createElement('div');
container.className = 'eveve-widget';
container.dataset.restaurant = '123';
container.dataset.theme = 'brand-roboto';
container.dataset.defaultGuests = '4';

// Add it to the DOM
document.getElementById('booking-section').appendChild(container);

// Initialize it
window.EveveWidget.init(container);

    Responsive Design

    The widget is fully responsive, but you can control its container width:

    <!-- Full width on mobile, 50% on desktop -->
<div class="booking-container" style="width: 100%; max-width: 500px; margin: 0 auto;">
    <div id="eveve-booking" data-restaurant="123"></div>
</div>

<style>
@media (min-width: 768px) {
    .booking-container {
        width: 50%;
        float: right;
        margin: 0 0 20px 20px;
    }
}
</style>

    Best Practices

    Performance Tip: Add the script at the end of your HTML body to avoid blocking page rendering.
    Important: Always use a unique ID for each widget container on the same page.
    • Use a CDN for faster loading of the embed script
    • Test your integration in different browsers and devices
    • Consider adding a loading placeholder before the widget initializes
    • Place the widget in a prominent location on your site for better conversion
    • Use the debug mode during development and disable it in production
